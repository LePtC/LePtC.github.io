“喂豆浆就卖萌～”

萌狸（MoeLe），全名：萌狸狸 PC。11942 年出生于狸星母星，父亲：萌大狸 K，母亲：萌芽狸 i。SHU 物理、数学、法学专业学生。

爱好：喜欢喝豆浆，尝遍星空市的所有豆浆店，能够根据豆浆的温度和甜度判断豆浆的来源、制作的时间等。如果不喝豆浆就会做饿梦，然后木有力气卖萌了…

话风：喜欢把「的」说成「滴」，喜欢加「的说」。

宠物：麦酱

工作：SHU 应用物理学计算密集型科研工作，同时业余时间喜欢收集自然界中的题猪进行科学研究，获得称号「虐猪狂人萌狸狸」。


{% include header.html param="豆浆雷达" %}

萌狸的天赋之一是豆浆雷达（Soya Radar）。萌狸的嗅觉异常灵敏，并在幼年的时候意外得到豆浆气味的熏陶，从此牢记并随时能准确识别豆浆的气味。正常情况下能以 $5 \sigma$ 置信度察觉百米范围内的豆浆源，在饿的时候灵敏度还将提升一个数量级。（在 s2m1 故事中，萌狸家距鲜饮店 1200 米）

但在有强烈刺激性气味的环境中，豆浆雷达的信号会被噪声压制，无法给出正确判断。


{% include header.html param="降落尾巴" %}

萌狸有一个大大的蓬松的尾巴，只要姿势正确的话，可以从普通建筑的高度安全地落到地面上。


{% include header.html param="晕动" %}

萌狸对频繁的急动度和科氏加速度有晕眩反应，表现为晕车、晕船、晕真空隧道、晕离心式人工重力场，但不会晕在近地大气层飞行的亚音速飞行器、运载火箭、加速式人工重力场。
