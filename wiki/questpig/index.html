Ph.D. -- pig has dream

{% include header.html param="题猪的生成" %}

题猪（Questpig），一种在自然界中天然生成的生物。狸神觉得这个世界太无趣了，说：要有脑洞，于是空间中便充满了量子猪场（Pigon field）。每当有人产生一个不靠谱的想法时，就有一只题猪从量子猪场中激发出来，飘荡在高维空间中。

{% include header.html param="题猪的形象" %}

题猪能够自行选择参与哪些与现实世界物质的相互作用，当它开始与现实世界产生相互作用时，会同时在现实世界产生一个可视的投影，这个投影就是人们所看到的题猪的形象。因为作者很懒，所以题猪的外形是一个三角形向外变形而成，然后安两个同样由三角形变形出来的耳朵，表情则就是颜文字 =(∞)=

题猪会保存少量生成它的题主的特征及性格。


{% include header.html param="题猪的目标" %}

题猪一生的追求就是看到生成它的那个不靠谱的想法的实现，因此大部分题猪都有极强的作死愿望。题猪为了达到目的，可做出超出目前物理学规律的事情，从而具有极高的科研价值。萌狸的工作就是收集自然界中的题猪，在可控的实验室环境下帮助题猪们实现作死愿望。


{% include header.html param="题猪的消失" %}

题猪在实现自己的愿望后，便会慢慢蒸发为量子泡沫，永久地从现实世界消失。

然而对于题猪来说，消失并不可怕，它们存在过的信息将留在现实世界中。永生才是世界上最可怕的事情，意味着它们愿望永远无法实现。

题猪的这种害怕生，追求死的人生观并不完全是一种黑白颠倒的人生观，用题猪自己的话来说，这是一种「朝闻道，夕死可矣」的人生观。
