我对待作品有两个特点，一：怕乱，怕到处都是，二：喜欢不停地改进。

我在线下是个特别致力于整理东西的人，从狸笔记大概能看得出来。于是在线上，我也决定用一个统一的网站来对我在网络上分享的所有资料进行汇总整理。

只有自己动手来写网站才能让自已完全满意，于是萌狸主页就诞生啦。在这里我将不受外界打扰，慢慢构建起我理想中的艺术品。

网站在达到满意的完善度之前将长期处于内部测试阶段。

{% include cheader.html param="版块规划" %} {% include header.html param="百科" %}

萌狸百科 是跨版块的说明汇总书，以词条的形式对作品中出现的概念提供背景说(剧)明(透)。

{% include header.html param="科幻" %}

萌狸科幻（尚未上线） 是一系列科幻故事，大致是讲述狸星人民是如何在一百个世纪间把狸星系的 26 颗行星给浪得只剩 12 颗的故事。

目前处于攒灵感阶段，最终攒成完整的作品可能还需几年的时间。

{% include header.html param="不科学" %}

萌狸不科学系列 是图文故事作品集，由于绘图工作量大，故明显比其它版块低产。

{% include header.html param="看不懂" %}

萌狸看不懂系列 是所有不脑洞的作品集，大部分为科普大杂烩，偶尔也画些有意思的图。

{% include header.html param="狸笔记" %}

狸笔记 是 LePtC 共享的物理、数学、外语等学科的笔记，记载了 LePtC 学过的知识及经验。

预计本科阶段的笔记的第一版将于 2015 年底完结，第一版以后还会更第二版第三版…永久更新下去。

{% include header.html param="项目" %}

萌狸项目（尚未上线） 是诸如 2048 physics 的一些有意思的项目，上线时间未定。
