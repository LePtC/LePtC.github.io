萌狸君对待作品有两个特点，一：怕乱，怕到处都是，二：喜欢不停地改进。

萌狸君在线下是一只特别致力于整理东西的强迫症，于是便决定在线上也做一个统一的网站，对所有在网络上分享的资料进行汇总整理。

对于一只强迫症，只有自己来写网站才能让自己完全满意的说… 于是萌狸主页就诞生啦，在这里偶可以不受外界打扰，慢慢构建起我理想中的艺术品～

{% include cheader.html param="版块规划" %} {% include header.html param="百科" %}

萌狸百科（内部测试）是跨版块的说明汇总书，以词条的形式对作品中出现的概念提供背景说(剧)明(透)。

{% include header.html param="科幻" %}

萌狸科幻（尚未上线） 是一系列科幻故事，大致是讲述狸星人民是如何在一百个世纪间把狸星系的 26 颗行星给浪得只剩 12 颗的故事。目前还处于攒灵感阶段。

{% include header.html param="故事" %}

萌狸故事（内部测试） 是图文故事作品集。萌狸君的绘画还处于练习阶段，可以预见这个版块会非常低产…

{% include header.html param="科普" %}

萌狸科普（内部测试） 是所有不脑洞的作品集，目前有酷炫动图，信息图示，看不懂系列等。

{% include header.html param="狸笔记" %}

狸笔记 是 LePtC 共享的物理、数学、外语等学科的笔记，记载了 LePtC 学过的知识及经验，将永久更新下去。

{% include header.html param="项目" %}

萌狸项目（尚未上线） 是诸如 2048 physics 的一些有意思的项目，上线时间未定。
