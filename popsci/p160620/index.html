根据能量守恒，石头落下的势能需大于将坂本抬升所需的势能，难道坂本比那块石头还轻？如果加上坂本有助跑且自己也可以出力弹跳，参考跳高的世界纪录约 2.4 米，弹上二楼倒不是不可能。动画中起码上的是？楼，所以坂本的体重一定是异于常人的轻。

{% include header.html param="用窗帘做降落伞" %}

坂本同学搞的这个降落伞啊，naive。如果你 DIY 过小降落伞，或观察过实际的降落伞，你会发现降落伞需要在中间留一个洞。你想想当气流都堆到伞下方时，你不在中间戳个洞，气流要怎么流出去？只能随机找个方向从一侧噗出去啦。然后你的降落伞就会在空中疯狂摇摆，最后拧成一坨，人就结结实实地摔地上啦。

{% include header.html param="旋转手臂恢复平衡" %}

什么？你以为坂本是靠手扇出风恢复直立的？你们呐还是图样。此处坂本显然是利用角动量守恒，靠手臂加速逆时针旋转来产生使身体顺时针转动的力矩恢复平衡的。
